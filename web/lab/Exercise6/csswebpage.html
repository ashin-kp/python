Cascading Style Sheets [CSS]

Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.CSS provides several methods to style paragraph elements (para tags) using different selectors. The most common approaches are using element selectors, class selectors, and ID selectors. Additionally, CSS offers advanced styling options for paragraphs. The text-transform property can change the case of text, such as converting text to uppercase, lowercase, or capitalizing the first letter of each word. The text-indent property can add an indent also be used to define reusable values, such as colors or spacing, which can be applied to paragraphs.


CSS PROPERTIES

accent-color Specifies an accent color for user-interface controls
align-content Aligns a flex container's lines within the flex container when there is extra space on the cross-axis
align-items Aligns flex items of the current flex line within the flex container
align-self Allows the default alignment (or the one specified by align-items) to be overridden for individual flex items
background-color Sets the background color of an element
border Sets the border around an element
border-radius Defines the radius of the element's corners
display Specifies the display behavior (the type of rendering box) of an element
